# EdLing Repository

This respository contains scripts for the analyses in the following publication:

Harmsen, W., van Hout, R., Cucchiarini, C., Strik, H. (2026). Automatic Measures of Oral Reading Fluency: Comparing Child and Adult Speech. [In review]

The corresponding data files are available on the Radboud Data Repository: 
Automatic Measures of Oral Reading Fluency: A Comparison between Child and Adult Speech
https://doi.org/10.34973/v8vy-1d71
